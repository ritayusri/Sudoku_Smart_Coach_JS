Sudoku Smart Coach
---------------------
Le Sudoku Smart Coach permet d'apprendre les règles et stratégies pour la résolution de différentes grilles de Sudoku

https://github.com/ritayusri/Sudoku_Smart_Coach_JS

---- Initiation du processus ----
Dézipper le répertoire: 
Dans le répertoire:
Double cliquer sur le fichier:
Index.html

---- Prerequis ----
Avoir installé sur son ordinateur un des fureteur suivant: Google Chrome, FireFox ou Opera
une connexion à internet pour jouer avec la version sur le web

---- Version Web du projet ----
Le jeu peu être utilisé en ligne à l'adresse URL: http://www-ens.iro.umontreal.ca/~yusririt/Sudoku_Smart_Coach_JS/index.html
